# Deciding-ensemble-ratio-using-OOF
캐글 대회에 참여할 때마다 여러 모델들 중 어떤 모델들을 뽑아 앙상블할지, 그리고 그 모델들을 어떠한 비율로 섞어 앙상블할지 정해야하는 순간을 마주했다. 
> ex) 최종 예측 값 = 0.6 * A 모델의 예측 값 + 0.4 * B 모델의 예측 값
>
Cassava 대회에 참여하며 Public Leaderboard Score만 믿고 위와 같은 0.6, 0.4의 수치를 정하게 되면 PB에 biased되는 성향을 가질 수 있음을 직접 경험할 수 있었다.
